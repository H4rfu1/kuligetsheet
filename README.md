# kuligetsheet
A python package for Grab Google Sheet data.

## Installation
Requirement library:
```bash
pip install csv
```

To install, run the following command:
```bash
python3 -m pip install kuligetsheet
```

## usage
Enter in python interactive mode:
```bash
python3
```
Using python package with following usage:
```python
import kuligetsheet
kuligetsheet.main()
```
